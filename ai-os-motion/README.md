# AI Marketing Operating System — Motion Graphic

30-second vertical product reveal (1080×1920) built with Remotion.

## Preview

```bash
cd ai-os-motion
npm run dev
```

Open composition **AIOperatingSystem**.

## Render

```bash
npm run render
```

Output: `out/ai-marketing-operating-system.mp4`
